来看看trait的总结整理

trait是服务于变量类型的，他规定了类型必须提供哪些功能语言特性

比如在之前要计算一个圆的面积，可以单独使用一个方法来解决，在得到圆的各个参数后，调用此方法就可以得出最后的结论

struct circle&#123;
    x: f64,
    y: f64,
    radius: f64,
&#125;

impl circle &#123;
    fn area(&self) -> f64&#123;
        std::f64::consts::PI * (self.radius * self.radius)
    &#125;
&#125;


fn main() &#123;
   let c1 = circle&#123;x:32.1,y:32.1, radius: 0.0 &#125;;
    println!("&#123;&#125;",c1.area());
&#125;

但是同样的我们也可以使用trait来实现这个原理，原理就是对circle来类型来创建一个trait

struct circle&#123;
    x: f64,
    y: f64,
    radius: f64,
&#125;

trait HasArea &#123;
    fn area(&self) -> f64;
&#125;

impl HasArea for circle &#123;
    fn area(&self) -> f64&#123;
        std::f64::consts::PI * (self.radius * self.radius)
    &#125;
&#125;

fn main() &#123;
   let c1 = circle&#123;x:32.1,y:32.1, radius: 1.0 &#125;;
    println!("&#123;&#125;",c1.area());
&#125;

乍一看感觉都差不多，并且trait的代码还多一点。但其实trait并不止这些

struct Cat &#123;&#125;
struct Dog &#123;&#125;

  fn talk(&self) &#123;
    println!(“meow”);
  &#125;
&#125;
impl Animal for Dog &#123;
  fn talk(&self) &#123;
    println!(“bark”);
  &#125;
&#125;
fn animal_talk(a: &dyn Animal) &#123;
  a.talk();
&#125;
fn main() &#123;
  let d = Dog &#123;&#125;;
  let c = Cat &#123;&#125;;
  animal_talk(&d);
  animal_talk(&c);
&#125;
