golang编译

golang对交叉编译的功能实现的很好
编译到linux上执行的程序

env GOOS=linux GOARCH=amd64 GOARM=7 go build

查看go的环境

  • GOROOT
    该环境变量的值为 Go 语言的当前安装目录。
  • GOPATH
    该环境变量的值为 Go 语言的工作区的集合（意味着可以有很多个）。工作区类似于工作目录。每个不同的目录之间用：分隔。工作区是放置 Go 源码文件的目录。一般情况下，Go 源码文件都需要存放到工作区中
    关于 IDE 新建 Go 项目。IDE 在新建完 Go 的项目以后，会自动的执行 go get 命令去把相应的基础包拉过来，
    如果没有使用IDE，则需要手动创建bin、pkg、src三个文件夹

helloworld的编译包

看一下helloworld程序的二进制文件
先写一个golang的helloworld的程序

package main

import "fmt"

func main() {
   fmt.Println("helloworld");
}

然后对其进行编译,go build ，再使用vim查看编译后的文件
设置查看形式为

:%!xxd

就可以查看到完整的二进制文件了
