Python os模块

在python中__file__ 表示当前正在使用的文件

获取文件所在路径

- 获取当前python程序的文件名
print(__file__)

- 获取当前文件的绝对路径，包括文件自己的名字
os.path.realpath(__file__)  # ==> print(__file__)

- 获取文件名字
os.path.basename(__file__) # 

- 获取当前文件的上级文件夹所在路径
os.path.dirname(__file__) 

- 获取当前程序文件所在项目的文件夹地址 
os.getcwd()   # 和os.path.dirname(__file__) 是有区别的

- 获取当前文件所在文件下的所有文件列表
os.listdir(os.path.dirname(__file__)) 

- 目录拼接
os.path.join(os.getcwd(),'hello.py')

- 获取上级路径
os.path.abspath(os.path.join(os.getcwd(), "../"))

- 获取上上级路径
os.path.abspath(os.path.join(os.getcwd(), "../.."))

实际使用

import os
print(__file__)
print(os.path.realpath(__file__))
print(os.path.basename(__file__))
print(os.path.dirname(__file__))
print(os.listdir(os.path.dirname(__file__)))
print(os.getcwd())
print(os.path.abspath(__file__))
print(os.path.join(os.getcwd(),'hello.py'))
print( os.path.abspath(os.path.join(os.getcwd(), "../")))
