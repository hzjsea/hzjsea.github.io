openPyXL模块

openPyXL模块是一个用来处理xls文件 ，也就是处理excel表格的python库

文档地址: https://openpyxl.readthedocs.io/en/default/tutorial.html

安装库

pip install openpyxl

使用

# 创建一个对象
from openpyxl import Workbook
wb = Workbook()

# 激活一个session
ws = wb.active
#or
ws1 = wb.crete_sheet("Mysheet")

# 设置title
ws.title = "New Title"

# 获取excel中所有的sheet names
wb.get_sheet_names()

操作完毕之后，表并没有保存，还需要调用save 来保存excel 

wb = Workbook()
wb.save("xxx")

加载已经存在的文件

wb = Workbook()
wb = load_workbook('old_excel.xlsx')

实战

一般情况下就这么用

# 1. 创建对象
# 2. 激活并添加表头
# 3. 传入data写入数据
def write_localtion_excel(path,data):
    wb = Workbook()
    ws = wb.active
    ws.append(['id','phone','date','time','data','name','position_name','detail','city','lng','lat'])
    for row in data:
        ws.append(row)
    wb.save(path)
