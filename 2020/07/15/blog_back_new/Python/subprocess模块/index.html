subprocess模块

subprocess模块

这里跟os里面的popen一起使用 两个都是调用子进程的库

popen调用子进程

import os
data = os.popen("ls -al").read()  # 直接返回终端打印出来的东西
data1 = os.popen("ls -al").readlines() # 返回一个数组
data2 = os.popen("ls -al").readline(10) # 返回一定数量的字符，由readline(size) 中的size决定
print(data,"\n",data1,"\n",data2)

但是popen会有一个问题，就是在调用awk的时候会出现错误,需要使用双花括号来解决 \{\{\}\}

import os
# res = os.popen("ls -al {} | awk '{ print $NF }'".format(".")).read() #  这样会报错无法识别{} 需要加双花括号来解决
res = os.popen("ls -al {} | awk '{{ print $NF }}'".format(".")).read()
print(res)

subprocess调用子进程

subprocess模块比较新把，意在替代一些老的模块，比如os.popen

filelist = subprocess.Popen(args="ls -al {} | awk '{{ print $NF}}' | xargs".format(filepath),shell=True,stdin=subprocess.PIPE,stdout=subprocess.PIPE)  # 这里返回的是一个tuple对象 (out,error)
(out,error) = filelist.communicate()

print(out)
