-C --icmptype жҢҮе®ҡicmpзұ»еһӢпјҲй»ҳи®Өзұ»еһӢдёәеӣһжҳҫиҜ·жұӮпјү
-K --icmpcode жҢҮе®ҡicmpзј–з ҒпјҲй»ҳи®Өдёә0пјү
вҖ“force-icmp еҸ‘йҖҒжүҖжңүICMPж•°жҚ®еҢ…зұ»еһӢпјҲй»ҳи®ӨеҸӘеҸ‘йҖҒеҸҜд»Ҙж”ҜжҢҒзҡ„зұ»еһӢпјү --icmp-gw й’ҲеҜ№ICMPж•°жҚ®еҢ…йҮҚе®ҡеҗ‘и®ҫе®ҡзҪ‘е…іең°еқҖпјҲй»ҳи®ӨжҳҜ0.0.0.0пјү
вҖ“icmp-ts зӣёеҪ“дәҺвҖ“icmp --icmptype 13пјҲICMPж—¶й—ҙжҲіпјү
вҖ“icmp-addr зӣёеҪ“дәҺвҖ“icmp --icmptype 17пјҲICMPең°еқҖжҺ©з Ғпјү
вҖ“icmp-help жҳҫзӨәICMPзҡ„е…¶е®ғеё®еҠ©йҖүйЎ№

UDP/TCPйҖүйЎ№

-s --baseport еҹәжң¬жәҗз«ҜеҸЈпјҲй»ҳи®ӨжҳҜйҡҸжңәзҡ„пјү
-p --destport зӣ®зҡ„з«ҜеҸЈпјҲй»ҳи®Өдёә0пјүпјҢеҸҜеҗҢж—¶жҢҮе®ҡеӨҡдёӘз«ҜеҸЈ
-k --keep д»Қз„¶дҝқжҢҒжәҗз«ҜеҸЈ
-w --win жҢҮе®ҡж•°жҚ®еҢ…еӨ§е°ҸпјҢй»ҳи®Өдёә64
-O --tcpoff и®ҫзҪ®еҒҮзҡ„TCPж•°жҚ®еҒҸз§»
-Q --seqnum д»…жҳҫзӨәTCPеәҸеҲ—еҸ·
-b --badcksum е°қиҜ•еҸ‘йҖҒдёҚжӯЈзЎ®IPж ЎйӘҢе’Ңзҡ„ж•°жҚ®еҢ…(и®ёеӨҡзі»з»ҹеңЁеҸ‘йҖҒж•°жҚ®еҢ…ж—¶дҪҝз”Ёеӣәе®ҡзҡ„IPж ЎйӘҢе’ҢпјҢеӣ жӯӨдҪ дјҡеҫ—еҲ°дёҚжӯЈзЎ®зҡ„UDP/TCPж ЎйӘҢе’Ң.)
-M --setseq и®ҫзҪ®TCPеәҸеҲ—еҸ·
-L --setack дҪҝз”ЁTCPзҡ„ACKпјҲи®ҝй—®жҺ§еҲ¶еҲ—иЎЁпјү
-F --fin дҪҝз”ЁFINж Үи®°set FIN flag
-S --syn дҪҝз”ЁSNYж Үи®°
-R --rst дҪҝз”ЁRSTж Үи®°
-P --push дҪҝз”ЁPUSHж Үи®°
-A --ack дҪҝз”Ё ACK ж Үи®°
-U --urg дҪҝз”ЁURGж Үи®°
-X --xmas дҪҝз”Ё X жңӘз”Ёж Үи®° (0x40)
-Y --ymas дҪҝз”Ё Y жңӘз”Ёж Үи®° (0x80)

еёёи§„йҖүйЎ№

-T --traceroute и·Ҝз”ұи·ҹиёӘжЁЎејҸ
вҖ“tr-stop еңЁи·Ҝз”ұи·ҹиёӘжЁЎејҸдёӢеҪ“ж”¶еҲ°з¬¬дёҖдёӘйқһICMPж•°жҚ®еҢ…ж—¶йҖҖеҮә
вҖ“tr-keep-ttl дҝқжҢҒжәҗTTLпјҢеҜ№зӣ‘жөӢдёҖдёӘhopжңүз”Ё
вҖ“tr-no-rtt дҪҝз”Ёи·Ҝз”ұи·ҹиёӘжЁЎејҸж—¶дёҚи®Ўз®—жҲ–жҳҫзӨәRTTдҝЎжҒҜ

ARSж•°жҚ®еҢ…жҸҸиҝ°(ж–°еўһеҠ зҡ„еҶ…е®№пјҢжҡӮж—¶иҝҳдёҚзЁіе®ҡ)
вҖ“apd-send еҸ‘йҖҒз”ЁжҸҸиҝ°APDзҡ„ж•°жҚ®еҢ…

    йҳІзҒ«еўҷжөӢиҜ•

    • -S иЎЁзӨәеҸӘеҸ‘йҖҒSYNеҢ…, д№ҹе°ұжҳҜжҸЎжүӢеҚҸи®®дёӯзҡ„иҜ·жұӮеҢ…
    • -c иЎЁзӨәеҸ‘еҢ…зҡ„ж•°йҮҸ
    • -a hostname иЎЁзӨәдјӘйҖ IPж”»еҮ»пјҢйҳІзҒ«еўҷе°ұдёҚдјҡи®°еҪ•дҪ зҡ„зңҹе®һIPдәҶпјҢеҪ“з„¶еӣһеә”зҡ„еҢ…дҪ д№ҹжҺҘж”¶дёҚеҲ°дәҶ
    • -p иЎЁзӨәзӣ®зҡ„з«ҜеҸЈ
      hping3 -S  -c 1000000 -a 10.10.10.10 -p 21 10.10.10.10

    з«ҜеҸЈжү«жҸҸ

    • -I eth0жҢҮе®ҡдҪҝз”Ёeth0з«ҜеҸЈ
    • -SжҢҮе®ҡTCPеҢ…зҡ„ж Үеҝ—дҪҚSYN
    • -p 80жҢҮе®ҡжҺўжөӢзҡ„зӣ®зҡ„з«ҜеҸЈ

    жӢ’з»қжңҚеҠЎж”»еҮ»

    • -I жҢҮе®ҡзҪ‘еҚЎ
    • -a дјӘйҖ ip
    • -S еҸ‘йҖҒsynеҢ…
    • -p жҢҮе®ҡзӣ®зҡ„з«ҜеҸЈ
    • -i жҢҮе®ҡеҸ‘йҖҒй—ҙйҡ”зҡ„ж—¶й—ҙ u1000иЎЁзӨә1000еҫ®з§’

    ж–Үд»¶дј иҫ“

    • вҖ“listen иЎЁзӨәзӣ‘еҗ¬жҢҮе®ҡеҶ…е®№,иҝҷйҮҢиЎЁзӨәзӣ‘еҗ¬signatureзҡ„еҶ…е®№
